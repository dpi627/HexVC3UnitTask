GitHub Spec Kit

Spec-Kit 是一套支援規格驅動開發的工具組，提供結構化指令，協助團隊定義需求、規劃實作、產生任務並運用 AI 流程確保專案品質。

Flow Diagram

Core Command

  • init: 從最新範本初始化新專案。
  • /speckit.constitution: 建立或更新專案的治理原則與開發指引。
  • /speckit.specify: 定義你想要建構的內容（需求與使用者故事）。
  • /speckit.plan: 以選定技術堆疊建立技術實作計畫。
  • /speckit.tasks: 產生可執行的任務清單以進行開發。
  • /speckit.implement: 根據計劃執行所有任務，完成功能開發。

Optional

  1. /speckit.clarify: 釐清描述不明或未規範的區域（建議於規劃前執行）。
  2. /speckit.analyze: 執行跨文件的一致性與覆蓋率分析。
  3. /speckit.checklist: 產生驗證需求完整性、清晰度與一致性的自訂檢查清單。
  4. check: 檢查已安裝工具與系統環境需求。
  5. 多寫了一項

Installation

uv tool install specify-cli --from git+https://github.com/github/spec-kit.git
